# 포스코 해외법인 로스터

사용자가 법인명(약칭)만 줘도 국가·유형을 식별하는 데 쓴다. **유형은 명칭(PC=Processing Center=가공센터)과 알려진 사실로 추정**한 값이다. "확인 필요"로 표시된 곳, 또는 사용자가 다르게 말하면 **사용자 입력을 우선**한다. 추정으로 분석을 진행할 때는 보고서에 "유형 추정 기준"을 각주로 남긴다.

유형 약어: **생산** = 생산법인(Mill, 제선·제강·압연·도금 등), **가공** = 가공센터(Coil/Processing Center, 절단·가공·유통)

## 유형 식별 규칙

1. 입력에 법인 유형이 명시되면 그대로 사용.
2. 없으면 이 로스터에서 매칭. 약칭 끝의 **PC**는 가공센터 신호.
3. 로스터에 없거나 "확인 필요"면 추정하지 말고 사용자에게 묻는다. 사용자가 "추정해서 진행"이라 하면 추정 근거를 각주로 남긴다.
