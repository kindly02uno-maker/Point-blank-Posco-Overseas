# 철강업 재무지표 기준 (Benchmarks)

철강업은 자본집약적이라 일반 제조업보다 부채 허용폭이 넓다. 아래 임계치는 철강업 통상 기준이며, 절대선이 아니라 **판정의 출발점**이다. 항상 추세(전년 대비)와 지표 간 관계를 함께 본다. 임계치를 기계적으로 적용하지 말고, 법인 유형·지역 여건을 감안해 해석을 조정한다.

판정 신호등: 🟢 양호 / 🟡 주의 / 🔴 위험

## 1. 재무안정성

| 지표 | 산식 | 🟢 양호 | 🟡 주의 | 🔴 위험 |
|---|---|---|---|---|
| 부채비율 | 총부채 / 자기자본 ×100 | ≤150% | 150~250% | >250% |
| 자기자본비율 | 자기자본 / 총자산 ×100 | ≥40% | 25~40% | <25% |
| 유동비율 | 유동자산 / 유동부채 ×100 | ≥150% | 100~150% | <100% |
| 당좌비율 | (유동자산−재고) / 유동부채 ×100 | ≥100% | 70~100% | <70% |
| 차입금의존도 | 총차입금 / 총자산 ×100 | ≤30% | 30~50% | >50% |
| 이자보상배율 | 영업이익 / 이자비용 | ≥3배 | 1~3배 | <1배 |

- **이자보상배율 < 1배**는 영업이익으로 이자도 못 갚는 상태로, 부채비율이 높지 않아도 단독으로 🔴 신호다.
- 부채비율이 높아도(🟡) 이자보상배율이 충분하고 차입구조가 장기·저리면 실질 위험은 낮게 본다. 반대로 단기차입 비중이 높으면 가중한다.

## 2. 수익성

| 지표 | 산식 | 🟢 양호 | 🟡 주의 | 🔴 위험 |
|---|---|---|---|---|
| 영업이익률 | 영업이익 / 매출 ×100 | ≥7% | 2~7% | <2% (또는 적자) |
| 순이익률 | 당기순이익 / 매출 ×100 | ≥4% | 0~4% | 적자 |
| ROE | 당기순이익 / 자기자본 ×100 | ≥8% | 3~8% | <3% |
| ROA | 당기순이익 / 총자산 ×100 | ≥4% | 1~4% | <1% |
| EBITDA마진 | EBITDA / 매출 ×100 | ≥12% | 6~12% | <6% |

- 철강 수익성은 철강 스프레드(제품가−원료가)에 크게 좌우돼 변동성이 크다. 한 해 수치보다 **2~3년 추세와 업황 사이클 위치**를 함께 본다.
- 가공센터는 구조적으로 영업이익률이 얇다(1~4%가 정상권). 가공센터에는 영업이익률 절대값보다 **가공마진 안정성·회전율**을 더 본다.

## 3. 활동성·운영 (확대여력 판단에 활용)

| 지표 | 산식 | 의미 |
|---|---|---|
| 총자산회전율 | 매출 / 총자산 | 자산 효율. 가공센터에서 특히 중요 |
| 재고자산회전율 | 매출원가 / 평균재고 | 낮으면 재고 적체·운전자본 부담 |
| 매출채권회전율 | 매출 / 평균매출채권 | 낮으면 대금회수 지연·현금흐름 압박 |
| 가동률 | 실생산량 / 생산능력 ×100 | 생산법인 핵심. 여유 있으면 증산 여력 |

## 4. 법인 유형별 강조 지표

### 생산법인 (Mill)
가동률 · 톤당 원가 · 부채비율·이자보상배율(투자 회수 구간) · EBITDA마진 · 설비투자 대비 현금창출. 환율·원료가·관세 민감도를 별도로 점검.

### 가공센터 (Coil Center)
운전자본 회전(재고·매출채권회전율) · 가공마진 · 총자산회전율 · 유동성. 자산이 가벼우므로 부채비율보다 **현금흐름과 회전**으로 건전성을 판단.

## 5. 판정 시 유의

- **단일 지표 단정 금지**: 부채비율 하나로 위험 결론 내리지 말 것. 안정성·수익성·활동성을 묶어 종합 등급을 매긴다.
- **추세 우선**: 임계치를 넘었는지보다, 좋아지는 중인지 나빠지는 중인지가 의사결정에 더 중요할 때가 많다.
- **결측치**: 계산 불가 지표는 공란·"N/A"로 두고 보고서에 명시. 임의 추정으로 채우지 않는다.
