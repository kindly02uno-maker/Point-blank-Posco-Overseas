# 입력 데이터 체크리스트

사용자 입력을 정리할 때 아래 항목을 확인한다. 없는 항목은 "N/A"로 두고 보고서에 명시.

## 법인 식별
- 법인명 / 소재 국가 / 법인유형(생산법인 · 가공센터) / 분석 기준연도

## 재무상태표 (B/S)
- 총자산, 유동자산, 재고자산, 매출채권
- 총부채, 유동부채, 총차입금(단기/장기 구분 가능하면)
- 자기자본

## 손익계산서 (P/L)
- 매출액, 매출원가, 영업이익, 이자비용, 당기순이익
- 감가상각비(EBITDA 산출용)

## 운영 지표
- 생산능력 / 실생산량 / 가동률 (생산법인)
- 가공능력 / 가공량 (가공센터)
- 판매량 (톤), 주요 제품 믹스
- 주요 수요처 / 전방산업

## 정성 — 대외 통상이슈
- 관세 / 반덤핑 / 세이프가드 / 수입쿼터 현황
- 환율 동향, 원료(철광석·석탄·모재) 조달 여건

## 정성 — 지역 특화 이슈
- 현지·역내 수요 전망, 전방산업 동향
- 경쟁 구도(현지·중국산 등), 물류, 환경·규제, 정치 리스크

## 추세 데이터 (가능하면)
- 위 정량 항목의 최근 2~3개년 시계열 → 추세·사이클 분석에 사용
